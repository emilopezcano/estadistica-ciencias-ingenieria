Capítulo 3 Ejemplo - Gráficamente

3.1 Ventajas e inconvenientes

  • No está afectada por valores extremos

  • Es más representativa de los datos

  • En variables simétricas, coincide con la media

  • Tiene muy malas propiedades matemáticas

3.2 La moda

  • Es el valor más frecuente

  • Puede no ser único

  • En variables continuas, intervalo modal

  • Se puede usar también en variables cualitativas

3.3 Ejemplo

3.4 Medidas de posición

  • El máximo, \(x_{max}\), y el mínimo, \(x_{min}\): nos informan de los extremos de los datos

  • Percentil \(P_p\), valor que deja por debajo de sí mismo el \(p\%\) de los datos, \(0 < p < 100\)

  • Los cuartiles: dividen los datos en cuatro partes:

    • \(Q_1 = P_{25}\)
    • \(Q_2 = P_{50}=Me\)
    • \(Q_3 = P_{75}\)

  • Análogamente, deciles y terciles

  • Cuantiles: como percentiles, expresados en tanto por uno, \(q_p, 0<p<1\)

3.5 Medidas de dispersión

  • Para caracterizar un conjunto de datos, además de las medidas de centralización y de posición, necesitamos medidas que nos den una idea de la dispersión de los datos.

  • Una media puede provenir de datos muy próximos a ella, o muy lejanos

3.6 El Rango

  • AKA recorrido
  • Diferencia entre el máximo y el mínimo

\[R = \max_i{x_i} - \min_i{x_i}\]

par(bg = "white")
plot(merma1, col="steelblue", pch=16)
abline(h=mean(merma1), lwd=2, col="darkred")
arrows(50, min(merma1), 50, max(merma1), code=3, col="darkred")

3.7 Varianza

  • Si tuviéramos una población con \(N\) valores \(X_i\) y media \(\mu\):

\[ \sigma^2= \frac{\sum\limits_{i=1}^N (X_i- \mu)^2}{N} \]

  • Para muestras de tamaño \(n\):

\[ s^2= \frac{\sum\limits_{i=1}^n (x_i- \bar{x})^2}{n-1} \]

par(bg = "white")
plot(merma1, col="steelblue", pch=16)
abline(h=mean(merma1), lwd=2, 
    col="darkred",
    main = "Merma proceso 1",
    xlab = "Pieza",
    ylab = "Peso")
for (i in 1:50){
  lines(x = c(i,i), y = c(mean(merma1), merma1[i]))
}

3.8 Desviación típica muestral

\[ s= \sqrt{\frac{\sum\limits_{i=1}^n (x_i- \bar{x})^2}{n-1}}= \sqrt{s^2} \]

  • AKA Cuasi-desviación típica (raíz de la cuasi-varianza)
  • Normalmente trabajamos con muestras
  • Es lo que el software calcula por defecto
  • Podemos calcular varianza poblacional como:

\[s_{poblacional}^2=s_{muestral}^2\cdot \frac{n-1}{n}\]

3.9 Cálculo abreviado

\[s^2= \frac{\sum\limits_{i=1}^n (x_i- \bar{x})^2}{n-1} = \frac{1}{n} \sum\limits_{i=1}^n x_i^2 - \bar{x}^2\]

3.10 Datos en tablas

3.11 Frecuencias absolutas

\[ s^2= \frac{1}{n-1} \sum\limits_{i=1}^n n_i x_i^2 - \bar{x}^2 \] ## Frecuencias relativas

\[ s^2= \sum\limits_{i=1}^n f_i x_i^2 - \bar{x}^2 \] En este caso resultaría la poblacional y habría que corregir para la muestral

3.12 Propiedades varianza y desviación típica

  • Muy sensibles a datos extremos
  • Entre la media y 2 desviaciones típicas \((\bar x-2s; \bar x + 2s)\), al menos el 75% de los datos (teorema de Chebycheff).
  • No mantiene la transformación lineal. Si \(Y=a+bX\) y la varianza de \(X\) es \(s_X\), entonces la varianza de \(Y\) es: \(b^2s^2_X\)

3.13 Tipificación

  • De las propiedades de la media y la varianza, si:

\[Z = \frac{X-\bar x}{s},\] entonces \(\bar z = 0; s = 1\).

  • Útil para comparar distintas escalas
  • Mantiene estructura correlación

3.14 Coeficiente de variación

  • La desviación típica está en las unidades de la variable

  • Si queremos comparar la variabilidad de dos variables, necesitamos una medida comparable

  • Coeficiente de variación:

\[CV = \frac{s}{\bar x}\] * También útil para comparar la misma variable en grupos distintos, cuando la media no es igual

3.15 Rango intercuartílico

  • Diferencia entre el primer y tercer cuartil
  • Rango del 50% de los datos centrales (alrededor de la mediana)

\[IQR = Q_3 - Q_1\]

3.17 Medidas de forma

  • Coeficiente de asimetría: Para ver si los datos son simétricos (mediana = media). Si no, tipo de asimetría.

  • Coeficiente de apuntamiento o curtosis: Cómo de “apuntados” son los datos (muy concentrados alrededor de la media)

3.18 Coeficiente de asimetría

\[\gamma_1 = \frac{m_3}{s^3}\] donde \(m_3 = \frac{1}{n}\sum\limits_{i=1}^n(x-\bar x ) ^3\)

  • \(\gamma_1 = 0 \implies\) Simétrica
  • \(\gamma_1 < 0 \implies\) Asimétrica negativa (valores bajos menos frecuentes que valores altos)
  • \(\gamma_1 > 0 \implies\) Asimétrica positiva (valores bajos más frecuentes que valores altos)

3.19 Coeficiente de asimetría 

m <- function(x, orden){
  sum((x-mean(x))^orden)/length(x)
}
set.seed(2)
gdata <- data.frame(Tipo = rep(c("Asimetrica_positiva", "Simetrica", "Asimetrica_negativa"), each = 100), 
                        Variable = 100 + c(rexp(100, 0.1),
                                     rnorm(100, 20),
                                     rbeta(100, 10, 0.5)))
ggplot(gdata, aes(Variable)) +
  geom_histogram(fill = "Orange", bins = 10, col = "white") +
  facet_wrap(~Tipo, scales = "free_x") +
  theme_bw()

3.21 Coeficiente de apuntamiento

\[\gamma_2 = \frac{m_4}{s^4}-3\] donde \(m_4 = \frac{1}{n}\sum\limits_{i=1}^n(x-\bar x ) ^4\)

  • \(\gamma_2 = 0 \implies\) Igual que la normal
  • \(\gamma_2 < 0 \implies\) Más aplastada que la normal
  • \(\gamma_2 > 0 \implies\) Más apuntada que la normal
